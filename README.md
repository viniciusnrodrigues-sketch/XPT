# Mapa de Escoamento EDF7

Pacote estatico pronto para GitHub Pages.

## Publicacao rapida

1. Crie um repositorio publico no GitHub.
2. Envie estes arquivos para a branch `main`.
3. Em `Settings > Pages`, selecione `Deploy from a branch`, branch `main`, pasta `/root`.
4. Acesse a URL publicada pelo GitHub Pages.

O arquivo principal e `index.html`. O mapa usa Leaflet/OpenStreetMap via CDN, entao a visualizacao publicada precisa de acesso a internet.
